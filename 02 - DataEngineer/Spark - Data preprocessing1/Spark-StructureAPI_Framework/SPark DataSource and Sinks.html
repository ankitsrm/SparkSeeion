SPark DataSource and Sinks

Data Store in

  • internal
    • HDFS Data storage
    • Cloud Based storage
  • external
    • JDBC Datasource
    • NoSql Data System
    • Cloud Data WareHouse
    • Stream Integrators

There are two approach to collect data:
- Bring data to Data lake - Store them to datalake distributed storage (Good for Batch Processing)
- Use Spark DataSource API to directly connect with External Data Source System (Good for Streaming Processing)
